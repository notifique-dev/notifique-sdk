# Security Policy

## Supported versions

Security fixes are applied to the latest code on `main`.

## Reporting a vulnerability

Please do not open public issues for security vulnerabilities.

Use GitHub private reporting:

- Repository -> Security -> Advisories -> Report a vulnerability

Or contact:

- suporte@notifique.dev

## Response expectations

- Initial acknowledgment: up to 72 hours
- Triage and severity classification: up to 7 days
- Fix and coordinated disclosure timeline depends on impact and affected SDKs
